# Shared Evaluation Scorecard

**Type:** MVP Product Design | B2B Procurement SaaS  
**Role:** Product Manager (Assignment)  
**Status:** Prototype Shipped

---

## Overview

Enterprise procurement teams evaluate 3rd-party vendors — software, IT services, consultants — by collecting input from 5-10 stakeholders across departments. Today this happens over emails and spreadsheets, leading to bias, delays, and misalignment that only surfaces at the end of the process.

This project is an end-to-end MVP design for a **Shared Evaluation Scorecard** feature for a B2B procurement SaaS product — enabling structured, asynchronous, collaborative vendor evaluation with early misalignment visibility.

---

## Problem in One Line

Procurement teams make high-stakes vendor decisions with misaligned stakeholder input collected through fragmented, manual processes.

---

## Solution

A collaborative scorecard feature where:
- Procurement managers create structured evaluations with custom criteria
- 5-10 stakeholders score vendors asynchronously against those criteria
- The system aggregates scores, highlights misalignment visually, and enforces time-bound deadlines
- A final decision summary is generated and shareable with leadership
